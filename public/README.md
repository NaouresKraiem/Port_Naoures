# Public Assets

## Profile Photo

To add your profile photo:

1. Place your photo in this folder (`public/`)
2. Name it `profile-photo.jpg`
3. Recommended specifications:
   - Format: JPG or PNG
   - Size: 800x800px or larger
   - Aspect ratio: Square (1:1) works best
   - File size: Under 500KB for optimal performance

If no photo is provided, the portfolio will automatically display your initials "NK" as a fallback with a gradient background.
